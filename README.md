# Lab_3-AlgsPacket

## Описание

Данный проект представляет собой CLI-консольное приложение миниоболочку которая позволяет вычислять функции фибоначи и факториал, реализовывать алгоритмы сортировки и действия со структурами данных, реализованную на Python в рамках **Лабораторной работы №2**

### Поддерживаемые возможности
#### Сортировки
* **пузырьковая сортировка** `sort bubble <nums...>`
* **Быстрая сортировка**`sort quick <nums...>`
* **Сортировка подсчётом**`sort count <nums...>`
* **Поразрядная сортировка**`sort radix <nums...>`
* **Блочная сортировка**`sort bucket <nums...>`
* **Пирамидальная сортировка** `sort heap <nums...>`
#### Числовые функции
* **Факториал (итеративный):** `fact <n>`
* **Факториал (рекурсивный):** `factr <n>`
* **Фибоначчи (итеративный):** `fib <n>`
* **Фибоначчи (рекурсивный):** `fibr <n>`
#### Стеки
* **Стек на списке (StackList):**
  * `stack push <x>`
  * `stack pop`
  * `stack top`
  * `stack len`
* **Стек на связанном списке (StackLinked) (тп)**  
* **Стек на очередях (StackQueue) (тп)**
#### Очереди
* **Очередь на списке (QueueList):**
  * `queue enqueue <x>`
  * `queue dequeue`
  * `queue front`
  * `queue len`
* **Очередь на связанном списке (QueueLinked),(функции такие же)**  
* **Очередь на стеках (QueueStack),(функции такие же)**
### Дополнительные возможности
* **Интерактивная CLI-оболочка:** удобный REPL-режим
